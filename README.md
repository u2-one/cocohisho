# Cocohisho - UTAU音声ライブラリ配布サイト

UTAU向けの無料音声ライブラリを配布するサイトです。

## 機能

- トップページから3つのライブラリページへの遷移
- 各ライブラリページでの世界観説明、キャラクター一覧、音源配布
- レスポンシブデザイン（スマホ対応）
- 音源情報の追加・更新が容易な構造

## 諸注意事項

- GitHub Pagesで公開するため公開リポジトリにしています。
- ページ内で利用しているアセットについては、著作元に権利があります。
- 基本的には配布用の立ち絵しか使っていないため画像のDL自体は問題ないですが、UTAU動画意外の用途に利用するのはNGです。
- もしバグや問題があれば[Twitter](https://x.com/_u2ONe) から連絡、もしくはプルリクエストをお願いいたします。
